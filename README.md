This is a cbt interface web app built by Osinachi Joshua. it allows regiser  for teachers and students alike, when teachers login, they can create exams based on various subjects and set the time interval for the exam and also the session year and then can upload photos with the questions which require pictures, and the options of the questions can either be multiple choice questions or a boolean true or false option and then the question can be set for the student to come on the platform and attempt the questions, when the student has finished witht the exams, the result is automatically sent to the teacher who set the exams making the teacher have access to all the records of students and the scores of those that have attempted his exam.
this web app was built using python flask framework

Installation guide for this web app
- git clone https://github.com/osinachi-2025/cbt-interface.git
- pip install -r requirements.txt (insure you already have the python environment downloaded already on your PC)
- python app.py
click on the localhost link that displays on the terminal for you.
